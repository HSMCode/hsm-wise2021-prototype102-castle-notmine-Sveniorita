# Ghosts

This is a one button game created in Unity. 

## Description

In this game you wander around a hanuted castle, meeting different kinds of ghosts on the way. Some are friendly, others not so much. 
In order to find your way out of the castle, you need to befriend some of the ghosts so that they may guide you.


## Getting Started

  - clone the repository
  - run the project in Unity


### Prerequisites

Unity version 2020.3.21f1
